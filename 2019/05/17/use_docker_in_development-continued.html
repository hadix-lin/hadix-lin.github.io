默认情况下，同一个主机上运行的docker容器会加入一个默认的bridge网络，可以通过容器的ip地址互相访问。但是镜像构建时是无法知道依赖容器的ip地址的，在上面的示例中是通过”mysql”，”redis”这样的hostname进行访问的，所以在此需要使用docker network create \ 来创建自定义网络，自定义网络会为加入该网络的容器提供自动DNS解析，容器可以通过名称互相访问。
